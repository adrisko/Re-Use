# Re-Use
Re-use est un jeu engager qui vise a l'apprentisage du recyclage pour tous le monde 
Le principe du jeu est simple, a chaque partie une multitude d'objet vous est proposée il faut choisir la bonne poubelle associée pour recycler l'object 
Ensuite grace aux objets recyclés de nouveaux objets peuvent etre crée.
